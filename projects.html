🔗 Mini Blockchain in C 🔽

A foundational blockchain simulation written in pure C as part of my Embedded Systems training at CDAC. It demonstrates how blockchain works at the lowest level — using structs, linked lists, and time-based block generation. No external libraries, no HAL — just clean, modular C logic.

  • ✅ Created Genesis block and chained subsequent blocks
  • 🕰️ Integrated timestamps and indexing
  • 🧠 Practiced struct design, memory management, and modular C
  • 🔧 Environment: Linux terminal, GCC
  • 📁 Code is open-source: GitHub Repository

Future Plans: Add SHA-256 hashing, Proof-of-Work, file persistence.
